# Local Recipe Finder & Instruction Maker (Mobile App)

Welcome to the mobile version of my **Local Recipe Finder** project! After building the web version, I transitioned into mobile app development to expand my skills. This app is built with **React Native** and **Expo**, connecting to a live backend API to help users discover recipes and follow cooking instructions on the go.

---

## 📱 Features

* **Recipe Discovery:** Find local recipes based on available ingredients.
* **Step-by-Step Instructions:** Detailed guides for every meal.
* **Cross-Platform:** Works seamlessly on both **iOS** and **Android**.
* **Live Backend:** Fetches real-time data from a custom API hosted on **Render**.

---

## 🛠️ Tech Stack

* **Framework:** [React Native](https://reactnative.dev/)
* **Development Tool:** [Expo CLI](https://docs.expo.dev/)
* **Language:** JavaScript / React
* **Backend:** Node.js / Express (Hosted on [Render](https://render.com/))
* **Networking:** Axios / Fetch API
